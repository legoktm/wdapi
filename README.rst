======================
Wikidata API
======================

.. image:: https://travis-ci.org/legoktm/wdapi.png?branch=master
  :alt: Build Status
  :target: http://travis-ci.org/legoktm/wdapi

This is a very simple library that parses Wikidata's "constraint" templates into machine readable output.

Installation
========

    pip install wdapi


Documentation
=========
